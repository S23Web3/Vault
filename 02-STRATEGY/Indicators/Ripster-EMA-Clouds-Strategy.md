## The 34/50 Cloud Rules (MOST IMPORTANT)

> **"The 34-50 EMA cloud decides overall trend UP or DOWN"**

### Trend Determination
| Price Position | Trend | Action |
|---------------|-------|--------|
| **ABOVE 34/50 cloud** | BULLISH | Only LONG |
| **BELOW 34/50 cloud** | BEARISH | Only SHORT |

### Critical Discipline Rule
> ⚠️ **If stock crosses over the cloud, LONG becomes SHORT and SHORT becomes LONG.**
> You MUST be disciplined. Many stocks change trend right on these clouds.

### Risk Management
- **34/50 cloud = Your RISK LEVEL**
- Whenever you long or short, the 34/50 cloud is where you place your stop

---

## The 5/12 Cloud Rules

> **"5-12 cloud is used to RIDE THE TREND or hit pops"**

### Entry Confirmation
| Signal | Action |
|--------|--------|
| 5 EMA crosses ABOVE 12 EMA | LONG confirmation |
| 12 EMA crosses BELOW 5 EMA | SHORT confirmation |

**Combine with:** Price must be on correct side of 34/50 cloud on 10-minute

### Trend Riding Rule
> Let trend ride as long as **10-min candle rides the 5/12 cloud**.
> When 10-min candle **closes under** the 5/12 cloud = **GET OUT**

## The 8/9 Cloud (Ribbon)

> **"8-9 cloud is used as PULLBACK position ADD"**

- Use for adding to winning positions on pullbacks
- Creates a mini ribbon for extra conviction
- Optional but helpful for scaling in

## Intraday Trading Rules

### LONG Rules
1. Only go LONG when Relative Strength or Stock trends **ABOVE the 34/50 EMA** on 10-min or 1-hour
2. Look for **higher highs / higher lows** on 10-min
3. 5/12 cloud cross confirms entry
4. Exit when 10-min candle closes below 5/12 cloud

### SHORT Rules
1. Only SHORT when price falls **BELOW the 34/50 EMA clouds** on 10-min or 1-hour
2. Look for **lower highs / lower lows** on 10-min
3. 5/12 cloud cross (12 under 5) confirms entry
4. Exit when 10-min candle closes above 5/12 cloud

### Entry Types

**5/12 EMA Bounce (Long)**
- Price pulls back to 5/12 cloud
- Cloud holds as support
- Enter on bounce

**34/50 EMA Bounce (Long)**
- Price pulls back to 34/50 cloud
- Cloud holds as support
- Enter on bounce with stop below cloud

**5/12 EMA Reject (Short)**
- Price rallies to 5/12 cloud from below
- Cloud acts as resistance
- Enter short on rejection

**34/50 EMA Reject (Short)**
- Price rallies to 34/50 cloud from below
- Cloud acts as resistance
- Enter short on rejection

## Swing Trade Rules

### Exit Rule
> Cut **full or 1/2 position** when price drops under the **34/50 Daily or 1Hr 20 EMA**

### Trend Confirmation
- 1-hour: Use 34/50 EMA for bullish (above) and bearish (below)
- 5/12 cloud on 1-hour works as a trendline as well

## Gap Trading Rules

### Gap Down Plays
| Scenario | Action |
|----------|--------|
| Price UNDER 10-min 50 EMA cloud + REJECTED | SHORT |
| Price moves ABOVE 10-min 50 EMA cloud | LONG |

### Gap Up Plays
| Scenario | Action |
|----------|--------|
| Price ABOVE 10-min 50 EMA cloud + HOLDS | LONG |
| Price falls BELOW 10-min 50 EMA cloud | SHORT (fade) |

**Exception:** Catalyst plays may override normal rules

## Volume Rules

> **"Volume is KEY - if stock has done 20% volume in first 30 mins, it will trend in same direction"**

### Volume Labels (TradingView)
- Updated Volume
- Average Volume
- Relative Volume (RVol)

### ATR/DTR Usage
- **ATR** = Average True Range (14-day average)
- **DTR** = Daily True Range (today's high - low)
- Compare ATR vs DTR to gauge volatility
- Higher values = increased volatility
- Use DTR to identify typical daily price range for targets

## MTF (Multi-Timeframe) Clouds

### Purpose
Overlay higher timeframe clouds on lower timeframe charts to see support/resistance levels while day trading.

### Recommended MTF Setup on 10-Min Chart
| MTF Cloud | Resolution | EMAs |
|-----------|------------|------|
| **1-Hour MTF** | 1 hour | 34/50 |
| **Daily MTF** | 1 day | 50/55 |

### How to Add (TradingView)
1. Add **2 separate MTF Cloud indicators**
2. First: Set Resolution = 1 hour, EMAs = 34/50
3. Second: Set Resolution = 1 day, EMAs = 50/55

**Principle:** Above clouds = bullish, below clouds = bearish (same as regular clouds)

## Complete Entry Checklist

### Pre-Trade
- [ ] 34/50 cloud confirms trend direction (above = long, below = short)
- [ ] 5/12 cloud cross confirmation
- [ ] Higher highs/lows (long) or lower lows/highs (short)
- [ ] Volume > 20% of average in first 30 mins
- [ ] Stop loss placed at 34/50 cloud

### During Trade
- [ ] 10-min candles riding 5/12 cloud
- [ ] No close against the 5/12 cloud

### Exit Triggers
- [ ] 10-min candle closes against 5/12 cloud
- [ ] Price crosses 34/50 cloud (trend flip)
- [ ] Target reached
