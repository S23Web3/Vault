# BBW Remaining Build: Coin Classifier + Ollama Review + CLI

## Context

BBW pipeline Layers 1-5 are complete and tested. Three components remain to make
the sub-project fully runnable end-to-end:

1. `research/coin_classifier.py` — KMeans volatility tier assignment per coin
2. `research/bbw_ollama_review.py` — Layer 6 LLM analysis of Layer 5 CSV output
3. `scripts/run_bbw_simulator.py` — CLI entry point wiring all layers together

**Delivery method**: One build script `scripts/build_bbw_remaining.py` using the
`build_staging.py` pattern (FILES dict → write → py_compile → run tests → report).
This avoids consuming all context tokens writing 9 files inline.
