## Key Learnings

### 1. Ollama Prompt Engineering
- **System prompts matter**: Explicitly state role and expertise
- **Temperature**: 0.3 for code (deterministic), 0.1 for reviews (strict)
- **Avoid example outputs in prompts**: Model will just return the example
- **Be explicit**: "Generate COMPLETE code, not placeholders"

### 2. Code Validation Pipeline
Order matters:
1. Syntax check (py_compile)
2. Structure check (imports, functions, classes)
3. Import test (actually load the module)
4. AI review (comprehensive security + quality)
5. Dependency mapping (requirements.txt)

### 3. Windows Automation
- `netplwiz` for auto-login
- `shell:startup` for startup scripts
- Task Scheduler for robust restart handling
- PowerShell `-NoExit` keeps window open for monitoring

### 4. GPU Management
- RTX 3060 (12GB) handles 30B model comfortably
- 56% utilization, 48% memory during generation
- nvitop provides better monitoring than nvidia-smi

## Notable Code Snippets

### Dependency Extraction
```python
def extract_dependencies(code: str) -> Set[str]:
    imports = set()
    for match in re.finditer(r'^import\s+([a-zA-Z0-9_]+)', code, re.MULTILINE):
        imports.add(match.group(1))
    for match in re.finditer(r'^from\s+([a-zA-Z0-9_]+)', code, re.MULTILINE):
        imports.add(match.group(1))
    return imports
```
